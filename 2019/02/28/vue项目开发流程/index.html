模块化设计

css模块化：设计原则是要有复用性，能够继承，并且完整，周期性迭代

设计方法：先从整体上考虑，再细化部分，比如移动端，先考虑布局样式，是用flex布局，还是grid布局，还是浮动布局，因为布局跟样式有关，可以把这些样式抽取出来做成模块，接着再考虑页面的方面，根据需求文档分析需要拆分成几个页面，找出这些页面样式相似的地方，写成可重复性，最后再细化到功能，一个功能能被多个页面使用，可以继承，像移动端每个页面左上角通常都有返回

js组件化设计 ：与css模块化相似，也是先整体后部分再颗粒化
